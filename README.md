# 🦠 COVID-19 Analytics Dashboard

## 📌 Project Overview

The COVID-19 Analytics Dashboard is an interactive data visualization project built using **Python**, **Streamlit**, **Pandas**, and **Plotly**. It allows users to explore COVID-19 statistics through interactive charts, filters, and summary metrics.

## 🚀 Features

* 📊 Dashboard Summary (Total Confirmed, Deaths, Recovered, Active Cases)
* 🌍 WHO Region Filter
* 📈 Top Countries by Confirmed Cases (Bar Chart)
* 🥧 WHO Region Distribution (Pie Chart)
* 📉 Confirmed vs Deaths (Scatter Plot)
* 📋 Interactive Dataset Preview

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly Express

## 📂 Dataset

* **country_wise_latest.csv**
* COVID-19 country-wise statistics including Confirmed, Deaths, Recovered, Active Cases, and WHO Region.

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/covid19-analytics-dashboard.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app:

```bash
streamlit run app.py
```
